Внутренний формат прикладной программы

Аудио

Сообщение

Для 3D моделей


model/example: (RFC 4735)

model/vrml: WRL файлы, VRML файлы (RFC 2077)

model/x3d+xml: X3D ISO стандарт для 3D компютерной графики, X3D XML файлы
