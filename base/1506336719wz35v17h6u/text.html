Через файл Шрифта ALS rub

Также можно воспользоваться шрифтом rouble

конвертируем шрифт через сервис Font Squirrel, в режиме «expert» отметив чекбоксы «No Subsetting» и «Style Linking».

Далее в CSS:

@font-face {
  font-family: 'rub';
  src: url('/font/rouble.eot');
  src: url('/font/rouble.eot?#iefix') format('embedded-opentype'),
       url('/font/rouble.woff') format('woff'),
       url('/font/rouble.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

.rub {
  font-family: 'rub';
}

HTML:

<span class="rub">a</span>

Для выбора стиля начертания, используются следующие символы данного шрифта(его нужно использовать в теге span класса rub):

  • a — Arial Regular
  • b — Arial Italic
  • c — Arial Bold
  • d — Arial Bold Italic
  • e — Georgia Regular
  • f — Georgia Italic
  • g — Georgia Bold
  • h — Georgia Bold Italic
  • i — Tahoma Regular
  • j — Tahoma Bold
  • k — Times Regular
  • l — Times Italic
  • m — Times Bold
  • n — Times BoldItalic
  • o — Lucida Sans
  • p — Lucida Sans Bold
  • q — Verdana Regular
  • r — Verdana Italic
  • s — Verdana Bold
  • t — Verdana Bold Italic
  • u — Futura
  • v — Trump Mediaeval
  • w — ITC Studio Script
